Open sublime text packages directory, and clone this repository into it

next, use the following command;

git clone https://github.com/buymeasoda/soda-theme/ "Theme - Soda" 

to install soda to sublime

Soda isn't mine, but goes great with it. All credit to the Soda developers



In user preferences, set as following;

{
	"color_scheme": "Packages/SublimeLimeTheme/Lime Hacker.tmTheme",
	"font_size": 11,
	"theme": "Soda Dark 3.sublime-theme"
}

After saving the preferences, you're good to go!
